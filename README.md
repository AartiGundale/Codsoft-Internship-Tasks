# Titanic Survival Prediction- Task 1
# Project Overview
This project analyzes Titanic passenger data to predict survival outcomes using SQL and Power BI.

# Tools Used
- SQL for data cleaning and feature engineering
- Power BI for interactive dashboard visualization

# Dashboard Highlights
- Survival rate by gender, class, and age group
- Age distribution of survivors

# How to Use
1. Import Titanic dataset from Kaggle into SQL.
2. Perform feature engineering (FamilySize, IsAlone).
3. Export cleaned dataset to Power BI.
4. Run Python scripts inside Power BI for logistic regression predictions.
5. Explore interactive dashboard.

## Movie Rating Prediction -Task 2
# Project Overview
This project predicts IMDb movie ratings based on features like genre, director, actors, budget, and revenue. Regression techniques are applied to estimate ratings and uncover factors influencing audience perception.

# Tools Used
- Python (Pandas, NumPy, Seaborn, Matplotlib)
- Scikit-learn (Linear Regression, Random Forest)
- Kaggle IMDb Movies Dataset

# Key Insights
- Votes and gross revenue strongly correlate with ratings.
- Genre and director influence ratings significantly.
- Regression model achieves R² ~0.65–0.75.

# How to Use
1. Download Kaggle movie dataset.
2. Clean and preprocess data (encode categorical features).
3. Train regression models (Linear Regression, Random Forest).
4. Visualize actual vs predicted ratings with Seaborn/Matplotlib.

## Sales Prediction- Task 3
# Project Overview
This project forecasts product sales based on advertising expenditure across platforms (TV, Radio, Newspaper). It demonstrates regression modeling and business insights for optimizing marketing strategies.

# Tools Used
- Python (Pandas, NumPy, Seaborn, Matplotlib)
- Scikit-learn (Linear Regression, Random Forest)
- Kaggle Advertising Dataset

# Key Insights
- TV + Radio spend has the strongest impact on sales.
- Newspaper spend shows weaker correlation.
- Regression model achieves R² ~0.85.

# How to Use
1. Download Kaggle Advertising dataset.
2. Perform EDA (correlation heatmap, scatter plots).
3. Train regression models to predict sales.
4. Visualize actual vs predicted sales.
