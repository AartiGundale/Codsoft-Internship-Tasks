# Codsoft-Internship-Tasks

## Project Overview
This project analyzes Titanic passenger data to predict survival outcomes using SQL and Power BI.

## Tools Used
- SQL for data cleaning and feature engineering
- Power BI for interactive dashboard visualization

## Dashboard Highlights
- Survival rate by gender, class, and age group
- Predictive model performance metrics
- Confusion matrix and accuracy KPIs

## How to Use
1. Download `titanic.csv` from Kaggle.
2. Open `Titanic_Survival.pbix` in Power BI Desktop.
3. Run the Python script inside Power BI to generate predictions.
